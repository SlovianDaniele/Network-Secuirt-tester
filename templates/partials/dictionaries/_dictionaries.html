Генерація словника

Генерація персоналізованого словника паролів на основі особистої інформації цілі. Інструмент використовує методологію CUPP (Common User Passwords Profiler) для створення словника, який включає комбінації імен, дат народження, назв компаній та інших особистих даних. Згенерований словник використовується для підбору паролів методом brute force під час тестування безпеки Wi-Fi мереж.
